# Essential Keyboard Shortcuts

Master these shortcuts to navigate Claude Code like a pro.

## Navigation & History

| Shortcut | Action | When to Use |
|----------|--------|-------------|
| `Esc Esc` | Rewind to previous checkpoint | Made a mistake? Undo the last agent action |
| `Ctrl+R` | Search prompt history | Find and reuse a previous prompt |
| `Up Arrow` | Previous prompt | Quick access to last prompt |

## Editing & Input

| Shortcut | Action | When to Use |
|----------|--------|-------------|
| `Ctrl+S` | Stash current prompt | Save what you're typing to finish later |
| `Tab` | Auto-complete | Complete file paths, command names |

## Mode Switching

| Shortcut | Action | When to Use |
|----------|--------|-------------|
| `Shift+Tab` (x2) | Enter Plan Mode | Complex tasks that need planning first |
| `Shift+Tab` (cycle) | Toggle modes | Switch between Auto/Plan/Manual |

## Special Commands

| Syntax | Action | When to Use |
|--------|--------|-------------|
| `!command` | Execute without AI | Run shell commands directly |
| `/command` | Slash command | Run a Claude Code command |
| `@filename` | Reference file | Include file content in context |

## Tips

### The Esc Esc Lifesaver

If Claude Code does something unexpected, double-tap `Esc` immediately. This rewinds to the last checkpoint before the action was taken.

### Prompt Stashing (Ctrl+S)

Typing a long prompt but need to check something first?

1. Press `Ctrl+S` to stash your current input
2. Do your research or run another command
3. Your stashed prompt will be waiting when you return

### Plan Mode for Big Tasks

Before starting a complex feature:

1. Press `Shift+Tab` twice to enter Plan Mode
2. Describe what you want to build
3. Review the plan before execution
4. Use `Shift+Tab` to exit Plan Mode when ready
