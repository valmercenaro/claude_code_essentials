# Beyond Coding

Claude Code isn't just for writing code.

## Productivity Management

Use Claude Code as your productivity assistant:

### Task Organization

```
Help me organize these tasks by priority:
- Fix login bug
- Write documentation
- Review PR #123
- Plan next sprint
- Update dependencies
```

### Meeting Prep

```
I have a meeting about [topic] in 30 minutes.
Help me prepare:
- Key points to discuss
- Questions to ask
- Background context I need
```

### Daily Planning

```
Here's what I need to do today:
[list of tasks]

Help me create a realistic schedule with breaks.
```

### Emails & Communication

```

### Documentation

```

### Reports

```
Create a progress report covering:
- What was accomplished this week
- Challenges encountered
- Next steps

Based on: @git log output or @specs/done/ folder
```

## Tips for Non-Coding Tasks

1. **Be specific about format**
   - "Create a bullet-point list"
   - "Write in paragraph form"
   - "Use a table format"

2. **Specify tone and audience**
   - "For a technical audience"
   - "In casual language"
   - "As if explaining to a beginner"

3. **Request iterations**
   - "Make it more concise"
   - "Add more detail to section X"
   - "Rewrite in a different tone"

4. **Use @ mentions for context**
   - Reference existing documents
   - Include relevant files
   - Provide examples to follow
