# handoff

## Purpose

Create a handoff document capturing the current session state for Module 10 work.

## Workflow

1. **Determine Handoff Number**
   - Check specs/handoffs/ for existing files
   - Use format: NNN-YYYY-MM-DD-topic.md
   - Increment NNN from last handoff

2. **Gather Context**
   - What was being worked on
   - What files were modified
   - Any pending tasks

3. **Create Handoff File**
   Save to specs/handoffs/NNN-YYYY-MM-DD-module10-work.md

## Template

```markdown
# Module 10 Handoff - [Date]

## Context
[What was being worked on]

## Completed
- [List of completed items]

## In Progress
- [List of in-progress items]

## Next Steps
1. [Next step 1]
2. [Next step 2]

## Key Files
- [Important file paths]
```
