## How to Install Skills

### From GitHub Repositories

```bash
# Clone the repository
git clone https://github.com/username/skill-repo.git

# Copy skills to your global folder
cp -r skill-repo/skills/* ~/.claude/skills/
```

### From .skill Files

```bash
# Unzip the .skill file (it's a zip archive)
unzip my-skill.skill -d ~/.claude/skills/my-skill/
```

### From Skills.sh

Visit https://skills.sh/, find a skill, and follow their installation instructions.
