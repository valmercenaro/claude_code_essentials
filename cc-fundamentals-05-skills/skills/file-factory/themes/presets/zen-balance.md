# Zen Balance Theme

**Character**: Calm, focused, meditative

## Color Palette
- Near White: `#F8F9FA` - Clean background (RGB: 248, 249, 250)
- Charcoal: `#212529` - Primary text (RGB: 33, 37, 41)
- Soft Gray: `#ADB5BD` - Secondary elements (RGB: 173, 181, 189)
- Accent Gray: `#6C757D` - Subtle accents (RGB: 108, 117, 125)

## Typography
- Headers: Georgia (serif for elegance)
- Body: Arial, Verdana

## Best For
Wellness, meditation, mindfulness, healthcare, minimal portfolios

## Effects
- Maximum whitespace
- Single large word as focal point
- Subtle gradients (cream, sage, gentle pinks)
- Minimal or no text shadows

## CSS Example
```css
.zen-balance {
  background: #F8F9FA;
  color: #212529;
}
.zen-secondary { color: #6C757D; }
.zen-muted { color: #ADB5BD; }
```
